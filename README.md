# some scripts

## Install

    git clone https://github.com/pashinin/scripts.git
    cd scripts
    make install

## Update (when you need)

    cd /usr/src/scripts
    sudo git pull
    make install

## Usage

Create a djvu file with `dir2djvu` command when in a dir with images.

Convert a zip-archive to 7z with `zip27z archive.zip` command.
