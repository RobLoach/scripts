# imgdir

## Install

        git clone https://github.com/pashinin/imgdir.git
        cd imgdir
        make install

## Update (when you need)

        cd /usr/src/imgdir
        sudo git pull
        make install

## Usage

1. Open a terminal in a directory with images
2. Run "dir2djvu" command
3. You have "dirname.djvu" from all images in the directory
